## Importing the Operating Systems public key

* Source: <https://osp4diss.vlsm.org/ETC/rmspubkey.txt>

```
[ -d $HOME/tmp/ ] || mkdir -pv $HOME/tmp/
cd $HOME/tmp/
rm -f rmspubkey.txt
wget -c https://osp4diss.vlsm.org/ETC/rmspubkey.txt
gpg --import rmspubkey.txt
gpg --list-keys

```
